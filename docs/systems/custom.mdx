Custom a basic nginx rewrite rule in place and runs different systems out of box. Some other systems will require a custom site.conf in the project folder.

Webroot is in `app/web` and most project use `app` as git folder. This allows to add sensitive information or documentation outside the webroot.

## Load

A custom load script can be added into your project folder.
