## Commands

### `console`

Shopware/Symfony CLI


## Initialize project

:::caution PHP 8.0
Installer is not PHP 8.0 compatible. Shopware itself is PHP 8.0 compatible.
:::

Run `make init` in an empty Docker Mate and follow the installation wizard. Afterwards use the web installer (no wrapper for the recovery cli at the moment).
